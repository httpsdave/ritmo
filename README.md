# ritmo

Discover and listen to live radio stations from around the world through an interactive 3D globe. Spin the globe, explore cities, and tune into thousands of stations in real-time.

Inspired by [Radio Garden](https://radio.garden/).

## What It Does

- **Interactive 3D Globe** — Navigate an Earth visualization with radio station markers
- **Browse by Location** — Click any city to see available stations  
- **Search** — Find stations, cities, or countries instantly
- **Live Streaming** — Play radio with full playback controls (play/pause, volume, skip)
- **Favorites** — Save your favorite stations for quick access
- **Responsive** — Works seamlessly on desktop, tablet, and mobile

## Tech Stack

| Layer       | Technology                                    |
| ----------- | --------------------------------------------- |
| Framework   | **Next.js 15** (App Router, TypeScript)       |
| 3D Graphics | **Three.js**, **React Three Fiber**, **Drei** |
| Styling     | **Tailwind CSS v4**                           |
| State       | **Zustand**                                   |
| Icons       | **Lucide React**                              |
| Data        | Radio Garden API (proxied via Next.js)        |
